# electron-twitter-spider
一款基于electron开发的推特、谷歌新闻采集软件。

- 支持批量采集
- 采集中自动截图
- 采集后自动排版生成doc文档
- 支持采集元素xpath路径配置
- 谷歌新闻自动分类发布源
